# tools/finalrecon.md

## Run FinalRecon with header + whois modules
    ./finalrecon.py --headers --whois --url http://example.com

## Common module flags (quick reference)
    --headers      # header information
    --sslinfo      # SSL certificate information
    --whois        # Whois lookup
    --crawl        # crawl target website
    --dns          # DNS enumeration
    --sub          # subdomain enumeration
    --dir          # directory enumeration
    --wayback      # Wayback URLs
    --ps           # fast port scan
    --full         # full reconnaissance scan

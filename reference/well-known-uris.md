# reference/well-known-uris.md

## Common `.well-known` URIs (quick reference)
| URI | Description | Status | Reference |
| --- | --- | --- | --- |
| `/.well-known/security.txt` | Contact info for reporting vulnerabilities. | Permanent | RFC 9116 |
| `/.well-known/change-password` | Standard URL to direct users to a password change page. | Provisional | w3c webappsec-change-password-url |
| `/.well-known/openid-configuration` | OpenID Connect discovery configuration. | Permanent | OpenID Connect Discovery 1.0 |
| `/.well-known/assetlinks.json` | Verify ownership of digital assets (e.g., Android app links). | Permanent | Digital Asset Links spec |
| `/.well-known/mta-sts.txt` | SMTP MTA Strict Transport Security (MTA-STS) policy. | Permanent | RFC 8461 |
